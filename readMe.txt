КРАТКОЕ ОПИСАНИЕ:
Проект "Формы" был создан с целью разработки веб-формы для регистрации новых аккаунтов в онлайн-магазине. 
Эта форма предоставляет пользователю возможность ввести необходимые данные для создания аккаунта. 

ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА:

index.html: Основной HTML-файл, содержащий разметку формы.
reset.css: Файл стилей для сброса стандартных браузерных стилей.
styles.css: Основной файл стилей, определяющий внешний вид формы.

ОСОБЕННОСТИ ПРОЕКТА:

* Реализация формы полностью соответствует макету и включает текстовые поля, чекбоксы для выбора уведомлений 
и поле с открывающимся списком опций для выбора страны.

* В проекте использована цветовая палитра, шрифты и изображения для создания приятного дизайна формы.

* Применение переменных в CSS для управления цветами и облегчения изменений в дизайне.

* Выравнивание текста и полей обеспечивает четкий и аккуратный внешний вид формы.

Проект доступен онлайн по ссылке https://form51.netlify.app благодаря публикации на платформе Netlify.

Дата создания проекта: март 2022 года.
